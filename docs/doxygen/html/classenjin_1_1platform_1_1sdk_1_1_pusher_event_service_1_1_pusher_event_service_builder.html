PusherEventServicePtr enjin::platform::sdk::PusherEventService::PusherEventServiceBuilder::Build ( ) const

Builds an instance of PusherEventService using the set parameters.

Returns
The service instance.
Exceptions
std::logic_errorThrown if the key is not set or if both the host and cluster are set at the time this method is called.
Remarks
If host is not set, then the service will default to ws-{cluster}.pusher.com and if the cluster is not set, then the cluster will default to mt1. If encryption is not set, then encryption will default to true.
